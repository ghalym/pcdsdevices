API¶

Selector Class¶

class pcdsdevices.sample_delivery.Selector(*args, **kwargs)¶

A Selector for the sample delivery system.

This class uses the M3 variant of selector, including two Sensirion liquid flowmeters, a Bronkhorst mass flow meter, and two 12-position Vici valves.

Parameters

  • prefix (str) – The base PV for the Selector.

  • name (str) – A name for the device.
